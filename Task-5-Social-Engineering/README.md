# Research Report on Social Engineering Attacks

This task explores common social engineering techniques used by attackers to manipulate individuals into revealing sensitive information or performing unauthorized actions.

The report covers:

* Phishing
* Spear phishing
* Vishing
* Smishing
* Pretexting
* Baiting
* Tailgating

It also discusses:

* Real-world impact
* Prevention strategies
* Security awareness best practices
